Operações Matemáticas em Java

Descrição
Este projeto foi desenvolvido como atividade acadêmica com o objetivo de aplicar conceitos de Programação Orientada a Objetos (POO) em Java.
O sistema realiza operações matemáticas básicas, como soma e divisão, utilizando herança, classe abstrata e tratamento de exceções.



Funcionalidades

Soma de dois números
Divisão de dois números
Tratamento de erro para divisão por zero

Conceitos aplicados

Classe abstrata
Herança
Polimorfismo
Exceções personalizadas
Try-catch

 Estrutura do projeto

 Divisao.java

 Via terminal:

 Exemplo de saída
